Get started with Jays

It's good that you're here, we can start using the library to create your website.

How to install Jays

We cannot start using the library without installing it in our project. So we will show you how to install it using the CDN.

In your <head> tag of your HTML file enter the following lines of code.

Note: The order of the tags is important.

<link rel="stylesheet" href="https://cdn.jays.dev/jays/1.0/jays.min.css">


<meta name="viewport" content="width=device-width, initial-scale=1.0">

And then at the end of your <body> tag enter this line of code:

<script src="https://cdn.jays.dev/jays/1.0/jays.min.js"></script>

If you've already entered all the lines of code in your HTML file and it doesn't work for you, we recommend that you read the documentation again or contact us.

And ready! You can now start using Jays in your project.

How to use Jays

The library is very simple to use.

First, you need to create a new <div> element and give it the class.

Example code:

<div class="container"></div>
